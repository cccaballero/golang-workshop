* Operadores
  :PROPERTIES:
  :CUSTOM_ID: operadores
  :END:
Golang soporta la mayoria de los operadores que tienen otros lenguajes.
Se dividen en dos categorias:

** Numericos
   :PROPERTIES:
   :CUSTOM_ID: numericos
   :END:
=+= => Addition

=-= => Subtraction

=/= => Division

=*= => Multiplication

=%= => Reminder

** Boleanos
   :PROPERTIES:
   :CUSTOM_ID: boleanos
   :END:
=!= => Not

=&&= => And

=||= => Or
